Neumorphism Glow Ring

映画「Harmony」の冒頭に現れる印象的なオブジェクトがモチーフ

ホワイトグレーを基調としたニュートラルな背景に、淡く発光するリングを配置。回転運動に応じてグローエフェクトが穏やかに脈動し、視覚的なリズムと存在感を与えています。

単なる装飾にとどまらず、「柔らかさ」と「デジタル感」の共存を意識して設計。ニューモーフィズム特有の陰影による奥行きと、トレース光の発光アニメーションを融合することで、ミニマルかつ印象的なインタラクションを実現しました。

Feature Tech

Vanilla JavaScript / CSS Animation
