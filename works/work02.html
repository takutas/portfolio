Slot Typing with Pink Glow

映画「Harmony」の冒頭に現れる印象的なタイプエフェクトがモチーフ

スロットマシンのような演出で文字が一文字ずつ確定していくタイピングアニメーションを実装。最終文字が確定すると全体がふわりと消えていく一連の演出により、**「出現」→「余韻」→「消失」**という視覚の物語性を意識しました。

全角・半角文字が混在する構成に対しては、各文字をラップする char-wrapper に data-type を付与し、それぞれに異なる幅・行間・文字サイズを動的に割り当てることで、整列と視認性を両立しています。

Feature Tech

Vanilla JavaScript / CSS Animation / Unicode-aware Layout
